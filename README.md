# App icon resizer

## About this project

Resize image to fit App Store app icon requirement.

## Framework Used

- Tauri
- React
- [icon-image](https://github.com/ElvisWong213/icon-image)

## Requirements

- Rust 1.76.0+
- Tauri 1.6.0+

## Setup

Run the command to start the app.
```
cargo tauri dev
```
