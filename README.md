## About this project

Convert your icon to different size for MacOS app icon

## Framework Used

- Tauri
- React
- [icon-image](https://github.com/ElvisWong213/icon-image)

## Requirements

- Rust 1.76.0+
- Tauri 1.6.0+

Run the command to start the app.
```
cargo tauri dev
```
